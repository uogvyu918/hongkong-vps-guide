## 实测感受与几个避坑提醒

**关于线路**

BGP 三网直连这个说法是真实的，但"直连"不等于"零延迟"。我从国内不同运营商实测过，电信延迟通常 30~40ms，联通 40~50ms，移动有时会到 60ms 上下。整体比美西 CN2 GIA 还是要快一截，但比纯香港本地机房的内网当然没法比。100Mbps 是 Fair Use 原则下的承诺带宽，长时间跑满会被限速，这一点所有"标 100Mbps"的香港 VPS 都一样，不是 ZgoCloud 一家的事。

**关于 IP 干净度**

ZgoCloud 给的是独立 IPv4，IP 段相对干净。但 IP 这种东西谁也不敢打包票，万一拿到手发现被某些服务拉黑了（比如 Netflix 解锁失败、某风控严的网站触发验证码），可以在客户后台提交 IP 更换申请，官方有 IP Change 的服务入口，价格不贵。这一点比某些"换 IP 要重装"的商家友好。

**关于退款**

特别提醒一句：**Specials 系列特价套餐官方明确写了不退款**——"No refunds/money back on those plans"。所以特价款下单前一定要想清楚，别买了之后说"线路不合适想退"。常规套餐的退款政策相对宽松一些，但具体条款以官方 TOS 为准，下单前花两分钟读一下。

**关于流量**

香港线统一 100Mbps，套餐之间差异主要是流量包大小——Lite 给 300GB，Starter 给 500GB，Standard 给 1TB。普通建站一个月用掉 100GB 都算多的，所以 300GB 对绝大多数人来说完全够。真要跑大流量业务（视频、CDN 源站、大文件分发），香港这台机器就不合适了，建议直接看 ZgoCloud 洛杉矶那边的 AMD VDS，年付 $96 起，给到 20TB 流量 + 1Gbps 带宽，跑量才是它的主场。
